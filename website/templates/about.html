CloudDelay Projesi Hakkında

CloudDelay, havacılık sektöründe yaşanan gecikme sorunlarına modern bir çözüm sunmak amacıyla geliştirilmiş kapsamlı bir veri mühendisliği projesidir. Amadeus API'den elde edilen gerçek zamanlı uçuş verileri, AWS cloud altyapısı kullanılarak işlenir ve makine öğrenmesi algoritmaları ile analiz edilir. Proje, yolculara seyahat planlaması konusunda önceden bilgi sağlarken, havayolu şirketlerine operasyonel iyileştirmeler için değerli içgörüler sunar.

Projenin Amaçları

Bu proje, havacılık sektöründeki veri odaklı karar verme süreçlerini desteklemek ve yolcu deneyimini iyileştirmek amacıyla tasarlanmıştır.

  • Amadeus API Entegrasyonu: Dünya'nın önde gelen seyahat teknoloji şirketlerinden biri olan Amadeus'un Flight Delay Prediction API'sini kullanarak gerçek zamanlı ve güvenilir gecikme tahminleri almak.
  • AWS Cloud Altyapısı: Amazon Web Services'in güçlü cloud hizmetlerini (S3, Glue, Athena) kullanarak ölçeklenebilir bir ETL (Extract, Transform, Load) süreci oluşturmak ve büyük veri setlerini verimli şekilde işlemek.
  • Veri Analitiği: SQL sorguları ve görselleştirme araçları (Tableau) ile derinlemesine analizler yaparak gecikme paternlerini keşfetmek ve operasyonel içgörüler sağlamak.
  • Kullanıcı Dostu Arayüz: Modern web teknolojileri kullanarak sezgisel ve responsive bir web uygulaması geliştirmek, böylece kullanıcıların kolayca gecikme tahminlerine erişebilmesini sağlamak.

Proje Geliştirme Süreci

Planlama ve Araştırma

Havacılık sektöründeki gecikme problemlerinin analizi, API araştırması ve teknoloji stack'in belirlenmesi. Amadeus API dokumentasyonunun incelenmesi ve AWS servislerinin planlanması.

Veri Toplama ve ETL

Amadeus API'den veri çekme script'lerinin yazılması, AWS S3'te veri depolama yapısının oluşturulması ve AWS Glue ile ETL pipeline'ının kurulması.

Veri Analizi ve SQL

AWS Athena ile SQL sorgularının yazılması, Tableau dashboard'larının tasarlanması ve veri kalitesi kontrollerinin yapılması.

Web Uygulaması Geliştirme

Flask web framework'ü ile backend geliştirme, Bootstrap ile responsive frontend tasarımı ve kullanıcı arayüzünün optimize edilmesi.

Test ve Deployment

Sistem testleri, performans optimizasyonları ve son kullanıcı testlerinin gerçekleştirilmesi. Dokümantasyon ve proje sunumunun hazırlanması.

Gelecek Geliştirmeler

CloudDelay projesi sürekli geliştirilmekte olup, gelecekte aşağıdaki özellikler eklenmeyi planlanmaktadır:
